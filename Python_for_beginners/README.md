## 왕초보를 위한 파이썬

* '왕초보'든 '초보'든 뭐든 간에, **모르는 것이 1개라도 있으면** 자만하지 않고 '연역적으로' 공부해야 한다.
  * 연역적 학습이란, **개념을 숙지하는 형태의 학습 방법** 이다.
  * 예를 들어 ```삼각함수 (sin, cos, tan)``` 를 단위원 + 직각삼각형을 이용하여 학습할 때, ```r```, ```x```, ```y```의 정의부터 정확하게 아는 것을 말한다.
* [Wikidocs 왕초보를 위한 파이썬](https://wikidocs.net/book/2)

* 본인이 **Python 문법을 충분히 숙지하지 못한** 이유
  * 학부에서 기본 문법을 배웠고, 연구 및 실무에서 결과물을 **무난히 구현** 은 할 수 있었음
  * 이로 인해 **Python의 문법을 추가적으로 학습** 하는 대신, **머신러닝 이론 역량 개발** 등 다른 부분 학습에 집중했음
* Python 문법 숙지의 필요성
  * 회사의 동료 개발자가 짠 Python 코드를 **한 줄도 빠짐없이 이해** 할 수 있음
  * 보다 Pythonic 한 코딩 가능
