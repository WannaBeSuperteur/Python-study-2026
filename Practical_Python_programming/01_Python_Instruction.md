## 1. 문자열 이스케이프 코드 (Escape code)

[1.4 문자열](https://wikidocs.net/84384)

* 이스케이프 코드 (escape code) 는 **제어 문자, 타이핑이 어려운 문자** 등을 나타낸다.

| 이스케이프 코드 | 설명                       | 용어 설명                           |
|----------|--------------------------|---------------------------------|
| ```\n``` | 라인 피드 (line feed)        | 커서를 **다음 줄로** 이동, 개행 (new line) |
| ```\r``` | 캐리지 리턴 (carriage return) | 커서를 **그 줄의 맨 앞으로** 이동           |
| ```\t``` | 탭 (tab)                  |                                 |
| ```\'``` | ```'```                  |                                 |
| ```\"``` | ```"```                  |                                 |
| ```\\``` | 백슬래시                     |                                 |

* 문자열 관련 함수

| 함수                | 설명                                    | 추가 설명                           |
|-------------------|---------------------------------------|---------------------------------|
| ```s.find(t)```   | s에서 t의 처음 등장 위치 (index)               | 존재하지 않는 값의 경우 **-1** 을 반환       |
| ```s.index(t)```  | s에서 t의 처음 등장 위치 (index)               | 존재하지 않는 값의 경우 **ValueError** 발생 |
| ```s.isalpha()``` | 문자열 전체가 영문자인지 여부                      |                                 |
| ```s.isdigit()``` | 문자열 전체가 숫자로만 구성되어 있는지 여부              |                                 |
| ```s.islower()``` | 문자열이 **소문자 1개 이상, 대문자 없음** 여부         |                                 |
| ```s.isupper()``` | 문자열이 **대문자 1개 이상, 소문자 없음** 여부         |                                 |
| ```s.rfind(t)```  | s에서 t의 처음 등장 위치 (index), **오른쪽부터 검색** | 존재하지 않는 값의 경우 **-1** 을 반환       |
| ```s.rindex(t)``` | s에서 t의 처음 등장 위치 (index), **오른쪽부터 검색** | 존재하지 않는 값의 경우 **ValueError** 발생 |

* 문자열 변환을 하기 위해서는 ```str``` 함수를 사용할 수 있다.
  * ```map``` 함수와 결합 시, ```map(str, iterable)``` 처럼 사용하여 **iterable 의 모든 원소를 string 으로 변환** 시킬 수 있다.

* 다양한 문자열

| 구분                    | 표현 방법           | 설명                                                                                   |
|-----------------------|-----------------|--------------------------------------------------------------------------------------|
| 원시 문자열 (raw string)   | ```r'string'``` | - 문자열을 입력한 그대로 (백슬래시 해석하지 않음) 해석<br>- **백슬래시가 중요할 때** 사용하면 좋음                        |
| 바이트 문자열 (byte string) | ```b'string'``` | - **8비트 바이트의 string** 으로, 저수준 입출력 (I/O) 에서 사용<br>- 인덱싱 시 **바이트 값을 정수로 반환** 한다는 것이 특징 |

## 2. 리스트 (list)

* 리스트는 **수학 연산을 위한 목적의 자료구조가 아님**
* 수학 연산 (행렬 등) 을 위해서는 **NumPy 등 별도 라이브러리** 를 사용하는 것이 좋음
